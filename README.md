# Society hub 

A community-driven web application that helps university students **discover, join, and engage with societies** across campus.

Societies can:
- Create and manage **events** (with student **RSVPs**)
- Post **announcements**
- Run **polls**
- Build community interaction around society activity

Students can:
- Explore societies by interests
- **Join** societies
- See a personalised feed + events list based on memberships

## Tech Stack (Module-aligned)
- Frontend: HTML, CSS, JavaScript, **Pug**
- Backend: **Node.js**, **Express**
- Database: **MySQL**
- DevOps: **Docker**, Git, **GitHub Actions**
