**Student Stories:**

**Task:** View List of Societies

As a student I want to view a list of available Societies within my University, so that I can see which ones to join. 

**Acceptance Criteria:**

* Finishes when all societies are displayed  
* Finishes when each Society shows at least name and description  
* Finished when all societies are linked to the profile page

**Task:** Search feature for Societies

As a student I want to search for societies by name or keyword so that I can quickly search societies that I am curious about.

**Acceptance Criteria:**

* Finishes when the student is able to enter text into the search field  
* Finishes when the search field text matches the results  
* Finishes when “no results show” appears 

**Task:** Join Societies

As a student I want to join a society so that I am able to get updates and maybe even join these events.

**Acceptance Criteria:**

* FInishes when student is able to click join  
* Finishes when the system processes his subscription   
* Finishes when user can go to MySocieties and sees the one he chose

**Task:** Society “Posts”

As a student I want to be able to view posts from society so that I am updated with announcements, events, and trips.

**Acceptance Criteria:**

* Finishes when notification for society post is shown  
* Finishes when user is able to view post  
* Finishes when user can differentiate by post types

**Task:** RSVP to Event Posts

As a student I want to RSVP to a society event so that I can confirm my attendance

**Acceptance Criteria:**

* Finishes when student can mark “Available”  
* Finishes when RSVP status saves  
* Finishes when student is told “You can only submit attendance once”

**Task:** User Profile Editing

As a student I want to be able to edit my email and manage my society memberships so that my personal information stays up to date with my needs.

**Acceptance Criteria:**

* Finishes when a student can update their email  
* Finishes when a student can make changes to there data base  
* Its done when student can leave societies and logout

**Organizer Stories:**

**Task:** Announcement Posts

As a society organiser I want to post announcements so that members are able to stay informed with any updates.

**Acceptance Criteria:**

* Finishes when organiser is able to create posts  
* Finishes when announcements are visible to the members  
* Finishes when posts display creation date

**Task:** Poll Posts

As a Organiser I want to create polls so that i can collect opinions for users

**Acceptance Criteria:**

* Finishes when an organiser is able to create a question and receive opinions   
* Finishes when users are able to vote  
* Finishes when votes are recorded and organised correctly

**Task:** Event Posts

As an Organiser I want to be able to create event posts so that members are able to attend organised events.

**Acceptance criteria:**

* Finishes when organiser is able to set title, date, location and description   
* Finishes when events appear on society page  
* Finishes when members are able to RSVP

**Task:** RSVP Lists on Event Posts

As a society organiser I want to view a list of members who RSVP’d so that I am able to plan their attendance correctly.

**Acceptance Criteria:** 

* Finishes when an organiser is able to see a list of attendees  
* Finishes when list updates after a new RSVP

**Task:** List of Society Members

As an organiser I want to see all members of my society so that I can manage and mitigate participation

**Acceptance Criteria:**

* Finishes when organiser can view a list of members  
* Finishes when member details include at minimum email and name

**Task:** Society Profile Page

As a student I want to view a society profile page so that I am able to understand what the society does and see recent activity on the page
